# Tipos de dados

Conforme o ECMAScript temos 9 tipos de dados:

* Data types
  * Primitive / Primitive value
  * Structural
  * Structural Primitive

## Primitivos 

**São tipos de dados imutáveis**

* String
* Number
* Boolean
* undefined
* Symbol
* BigInt

## Estruturais 

* Object
  * Array 
  * Map
  * Set
  * Date
  * ... 
* Function

## Primitivos Estruturais / Structural Root Primitive

* null <!-- objeto vazio -->
